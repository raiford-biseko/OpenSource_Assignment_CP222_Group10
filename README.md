# Security Incident Reporting System

## Project Title
Security Incident Reporting System

## Degree Program
Cybersecurity and Digital Forensic Engineering (BSc CSDFE)

## Group Number
Group 10

## Project Description
This is a web-based Security Incident Reporting System developed using PHP.
It allows users to record security incidents, display incident reports,
and search for incidents by incident ID. The system is developed as part of
the CP 222 Open Source Technologies course at the University of Dodoma (UDOM).
