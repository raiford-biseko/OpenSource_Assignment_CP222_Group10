# Security Incident Reporting System

## Project Title
Security Incident Reporting System

## Degree Program
Cybersecurity and Digital Forensic Engineering (BSc CSDFE)

## Group Number
Group 10

## Project Overview
This is a web-based Security Incident Reporting System developed using PHP and MySQL.
It allows users to record security incidents, display incident reports,
search for incidents by incident ID, and delete incidents.
The system is developed as part of the CP 222 Open Source Technologies course
at the University of Dodoma (UDOM).

## Installation Steps
1. Install Apache, PHP and MariaDB on your system
2. Start Apache and MariaDB services
3. Create a database named incident_db
4. Create a table named incidents with the provided schema
5. Copy the project files to /var/www/html/incident_system/
6. Open browser and go to http://localhost/incident_system/
